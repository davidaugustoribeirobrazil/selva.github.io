SELVA — Contract Ownership Renounced

Date: October 27, 2025 (UTC−3)

We have renounced ownership of the SELVA token contract on Base. The token is now permissionless/immutable (owner() returns 0x000…0000).

Contract (Base): 0x5bD472E9c0fE7A6986Bc8E661BBc092f716133f2

On-chain proof:

  • Timelock schedule: 0xd8db…357e
  • Timelock execute: 0x0a6d…2720
  • owner() reading: BaseScan Read Contract

This change improves user safety by removing admin keys. There are no remaining privileged admin functions under key control (e.g., fee/blacklist toggles).
